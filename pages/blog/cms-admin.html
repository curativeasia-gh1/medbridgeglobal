How to deploy to medbridge-global.com

The live site runs from the GitHub repository curativeasia-gh1/medbridgeglobal. To add the blog:
  1. Download blog.html (index) and post.html (single post template) from this package
  2. In the repo, create the folder pages/blog/
  3. Upload blog.html → rename to index.html inside pages/blog/
  4. Upload post.html → keep as post.html inside pages/blog/
  5. Each time you write a new post below, click Generate Post File — this creates a standalone slug.html file to drop into pages/blog/
  6. Update the shared POSTS array in both index.html and post.html with the new post's metadata
  7. Push to GitHub — the site will update automatically

Generate a post HTML file

Select a post, then click Generate. Copy the output and save it as pages/blog/[slug].html in the GitHub repo.

Export POSTS data array

Copy this JSON and paste it into the const POSTS = [...] in both index.html and post.html to update the listing and routing.
